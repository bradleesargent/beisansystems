Cabin filter housing removal

Remove cabin filter housing lower cable bracket cover. Pry off cover clips and remove cover (flathead).


Disconnect 3 cabin filter housing cover toggle fasteners by turning 1/4 turn counterclockwise.
Pull cover forward and up to remove.

Remove air filter.


Disconnect cables from cabin filter housing cable bracket clips.


Remove cabin filter housing 4 mounting bolts (T30 torx bit socket 3/8� / 3/8� ratchet & extension).


Pull cabin filter housing forward and out of engine bay.

Cabin filter housing installation


Mount cabin filter housing onto engine bay bulkhead.
Install 4 cabin filter housing mounting bolts (T30 torx bit socket 3/8� / 3/8� ratchet).

Install cables onto cabin filter housing lower clips.


Insert filter into cabin filter housing.


Mount cabin filter housing cover.
Install 3 cabin filter housing cover toggle fasteners by pressing in and turning 1/4 turn clockwise.

Install cabin filter housing lower cable bracket cover. Press on cover until clips snap in place.
