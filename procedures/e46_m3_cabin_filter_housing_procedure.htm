Cabin filter housing removal


Disconnect cabin filter housing 3 cover toggle fasteners by turning 1/4 turn counterclockwise.
Pull cover forward and up and remove.


Pull out and remove air filter.
Note: Air filter dirty side is down.


Remove cabin filter housing cable bracket cover.
Pry off cover top and then bottom clips and remove cover (fingers).

Remove cables from cabin filter housing cable bracket.


Remove cabin filter housing 4 mounting bolts (T30 torx bit socket 1/4� / 1/4� ratchet & extension).


Pull cabin filter housing up, forward, and out of engine bay.

Cabin filter housing installation


Mount cabin filter housing onto engine bay bulkhead.
Move hoses forward to allow mounting.
Insert cabin filter housing into position while lifting up hood seals at each front corner of housing to allow insertion.
Align cabin filter housing bolt holes with bulkhead bolt holes.
Position hood seals at cabin filter housing front corners onto housing seal, and cabin filter housing seal ends onto bulk head seal (picture).


Install cabin filter housing 4 mounting bolts (T30 torx bit socket 1/4� / 1/4� ratchet).


Install cables into cabin filter housing cable bracket.
First install smooth cable in bracket lower slot, then install corrugated cable in bracket upper slot.
Press each cable until it snaps in all along cable bracket.


Install cabin filter housing cable bracket cover.
Install cover lower clips onto bracket, then press cover upper clips onto bracket.
Press upper and lower cover clips onto bracket until they snap into place.


Insert filter into cabin filter housing.
Note: Filter dirty side is down.


Mount cabin filter housing cover.
Insert cover under bulkhead seal and fully press in.
Install cabin filter housing 3 cover toggle fasteners by pressing in and turning 1/4 turn clockwise.
